# NVIM CONFIGURATIONS

1. Fist clone this repo
2. Prerequisites before running ``nvim`` in terminal  
    - if windows then need to install "curl" from your package manager 
      example: choco,scoop
    - need to have a c compier in system 
    - ripgrep (this is used for `live grep` feature)
    - need to have rust installed to use cargo
    - install `stylua` which is prefered for liniting lua 
    - install node and install prettier (used for liniting)
    - install python and install black (used for liniting)
    - install nerd Fonts 
3. run `nvim` in terminal it'll download plugins using package manager


NOTE: This is how i prefer my nvim config to be
